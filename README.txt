ReNew Notebook Store - Solution 2 adapted project
Creators: Berenice Olawale and Meriem Ben Zouitine

Setup in XAMPP:
1. Copy this folder to xampp/htdocs.
2. Start Apache and MySQL.
3. Open phpMyAdmin and import database/renew_notebook.sql.
4. Check includes/config.inc.php if your MySQL password is not empty.
5. Open http://localhost/renew_solution2_notebook_project/ in your browser.
